# SiNtax Antivirus

SiNtax is a free antivirus for Windows designed to protect the system from malicious software in real time. The program provides a wide range of functions, including process monitoring, virus scanning, threat detection, and system recovery.

---

## Main functions

- Real-time monitoring of system processes and resources
- Hidden monitoring (even when the Task Manager is closed)
- Detection of masking processes
- CPU, GPU, RAM consumption and component temperature
- Monitoring network connections and disk activity
- Fast and complete system scan
- Signature and heuristic scanning
- Detectors of cryptominers, ransomware, Trojans, rootkits and spyware
- Restore system files, registry and auto-upload
- Quarantine for suspicious files
- Emergency tools to block threats
- Customizable interface with dark and light theme

## Installation

1. Download the latest release from the section [Releases](https://github.com/AuriXDev/SiNtax/releases ).
2. Run the SiNtax_Setup installer.exe` on behalf of the administrator.
3. Follow the instructions of the installation wizard.
4. After installation, the antivirus will start automatically.

---

## Usage

- To start the scan, click the "Scan" button on the home screen.
- To configure the scan settings, go to the Settings section.
- Use the "Monitoring" tab to view active processes and threats.
- In the "Quarantine" section, you can manage isolated files.
